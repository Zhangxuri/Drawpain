这是一个java语言编写的简易的绘图板。

能够实现曲线、直线拖拽、文件保存和打开、正方形（三角形）等绘制。

可以调整线条粗细，可以调整为虚线，可以更改显色，可以实现橡皮擦。

可以实现撤销。

主要供初学者学习。

# 基于JAVA不知道几实现的画图版

### 功能
- [x] 画直线  
- [x] 画椭圆
- [x] 画矩形
- [x] 画笔
- [x] 虚线画笔
- [x] 画笔颜色
- [x] 画笔粗细
- [x] 保存
- [x] 读取
- [x] 画45°角直线
- [x] 画正方形
- [x] 画圆
